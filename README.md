\# 深汕天子山树木认养系统



本仓库为《管理信息系统开发方法》课程期末项目仓库，用于统一管理项目代码、数据库资料、接口文档与协作说明。



\## 项目简介

本项目面向深汕天子山场景，开发一个“树木认养系统”，实现用户认养树木、查看我的树、查看摄像头抓拍/监控，以及管理员进行树木管理、订单管理、养护记录管理等功能。
