七、联调过程中已修复的问题
1. 用户端认养大厅筛选无效
原因：后端 /api/tree/list 未按 status 参数过滤
处理：已修改 backend-b/app.py
状态：✅ 已修复

2. 用户端“我的树”页面不显示数据
原因：前端将 res.data 直接当数组使用，但真实返回结构为 res.data.list
处理：已修复前端读取逻辑
状态：✅ 已修复

3. 用户端视频播放器 src 传值错误
原因：前端把 RTSP 接口返回对象直接传给播放器
处理：改为从返回对象中提取字符串字段（如 playUrl / url / rtspUrl）
状态：✅ 已修复

4. 管理端树木新增/编辑后所属公司为空
原因：树木表单未传 companyId
处理：新增“所属公司”下拉框并回填
状态：✅ 已修复

5. 管理端养护记录树种显示为空
原因：后端 maintenance/list 未联表返回 treeType
处理：后端已联表 tree 表补充字段
状态：✅ 已修复

6. 管理端摄像头状态最后检测时间为空
原因：后端返回字段名 checkTime，前端读取字段为 lastCheck
处理：后端字段统一为 lastCheck
状态：✅ 已修复

7. 公司管理 / 养护记录显示异常测试数据
原因：当前 SQLite 数据库中存在脏数据
处理：已直接修改 .db 中异常记录
状态：✅ 已修复
