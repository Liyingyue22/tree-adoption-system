# frontend-admin

天子山树木认养系统 - 管理端前端

## 技术栈
- Vue3
- Vite
- TypeScript
- Axios
- Element Plus
- Pinia

## 已完成功能
- 管理员登录
- 登录守卫
- 退出登录
- 树木管理
- 订单管理
- 摄像头状态
- 公司管理
- 养护记录管理

## 启动方式

```bash
npm install
npm run dev
