# frontend-admin

天子山树木认养系统 - 管理端前端

## 技术栈
- Vue3
- Vite
- TypeScript
- Axios
- Element Plus
- Pinia

## 已完成功能
- 管理员登录
- 登录守卫
- 退出登录
- 树木管理（列表 / 新增 / 编辑 / 删除）
- 订单管理（列表）
- 摄像头状态（列表）
- 公司管理（列表 / 新增 / 编辑 / 删除）
- 养护记录管理（列表 / 新增）
